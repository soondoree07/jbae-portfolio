### 2024년 작품 배치 조정
- 나비 정원을 깊은 곳 어딘가 아래로 이동
- 깊은 곳 어딘가 ↔ 풀밭에서 좌우 배치 (pair-1)
- 밤이 지나면 ↔ 봄바람 흩날리며 좌우 배치 (pair-2)
- 모든 것의 역사 compact, 바람의 속삭임 large, 숨겨진 의미 compact 적용

### 작품 크기 비례 레이아웃
- 물리적 cm 크기 → 화면 비례 표시 (1cm = 6px, 320~1050px)
- 좌우 배치 작품은 실제 가로 cm 비율로 flex-basis 계산
- 그룹 내 flex-basis 버그 수정: d.w(세로) → d.h(가로)

### 갤러리 레이아웃 4열 그리드로 전면 개편
- flex → CSS grid 4열
- 반응형: 1024px↓ 3열 / 768px↓ 2열 / 480px↓ 1열
- 비례 사이징 및 그룹 래퍼 제거
